# Calculadora Básica en Python

Esta es mi primera calculadora hecha en Python.

La desarrollé mientras comenzaba a aprender programación, siguiendo un tutorial como base y utilizándolo para entender cómo funciona realmente el código. A medida que avanzaba, fui agregando pequeñas mejoras y detalles propios con las cosas que iba aprendiendo en el camino.

El proyecto incluye operaciones básicas como:

* suma
* resta
* multiplicación
* división

También agregué validaciones simples, manejo de errores y algunos detalles visuales en consola para practicar y experimentar un poco más allá del tutorial.

Hice esta calculadora principalmente por diversión, aprendizaje y para empezar a relacionarme con el código de una manera práctica. Más que hacer algo perfecto, quería construir algo que me ayudara a entender la lógica detrás de la programación y recordar el inicio de este proceso.

Probablemente en el futuro mire este proyecto y vea muchas cosas que mejorar, pero justamente por eso quise guardarlo aquí: como un recuerdo del comienzo.
